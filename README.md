### 1. Clone the Repo

```bash
git clone https://github.com/xevrion/chat-in-home.git
cd chat-in-home
```

### 2. Set Up the Server

```bash
cd server
npm install
```

Create a `.env` file in the `server/` directory:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

Start the server:

```bash
npm start
```
